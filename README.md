A simple React-based website that allows users to explore various places across India, view brief information about each destination, and get an estimated idea of the cost required to visit them.

## About the Project

This project was built to practice **React fundamentals** by creating a real-world style UI.  
It focuses on displaying structured data, rendering components dynamically, and building a clean, user-friendly interface.

## Tech Stack

- **React**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS**

## Features

- List of popular travel destinations in India
- Short description for each place
- Approximate travel cost for visiting each destination
- Clean and simple UI built with React components
- Responsive layout for different screen sizes

## Concepts Covered

- React functional components
- Props and component reusability
- Rendering lists using `map()`
- Basic state management
- Clean project structure
